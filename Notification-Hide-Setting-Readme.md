# 通知無効化オプションの強制キャンセル機能

> [!NOTE]
> MacroDroid などの自動化アプリと組み合わせて使う上級者向け機能です。

> [!IMPORTANT]
> これは LIMEs の設定画面には存在しない隠し機能です。

---

## 概要

「特定のユーザー以外からの通知を無効化」オプションを有効にしている状態でも、**制御ファイルを配置することでキャンセルし、全員からの通知を受け取れます。**

MacroDroid などで制御ファイルの作成・削除を自動化することで、時間帯や状況に応じた通知管理が可能になります。

---

## 制御ファイル

```
{設定フォルダ}/Setting/Notification_Setting_False.txt
```

> [!TIP]
> ファイルの中身は不問です。空ファイルでも動作します。

| 制御ファイル | 動作 |
|-------------|------|
| **あり** | 通知無効化オプションをキャンセル → 全員から通知を受け取る |
| **なし** | 通常通り（特定ユーザー以外の通知はOFF） |

- バックグラウンド監視のため、**LINE の再起動不要でリアルタイムに反映**される

---

## MacroDroid での使用例

> [!TIP]
> ファイル操作は MacroDroid の「ファイルを作成」「ファイルを削除」アクションで実現できます。

| トリガー例 | アクション |
|-----------|-----------|
| 特定の時間帯（例：就寝前） | ファイルを削除して通知無効化を再有効化 |
| 特定のWi-Fiに接続したとき | ファイルを作成して全通知を受け取る |
| 充電開始時 | ファイルを作成して全通知を受け取る |
