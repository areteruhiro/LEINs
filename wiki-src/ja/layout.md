# レイアウト・UI

[ホームへ戻る](../index.html) / [設定画面の使い方](settings-ui.md)

ボトムバーの表示、アイコン配置、タップ範囲を調整します。

> 2026年6月21日時点の `LEINsOptions.java`、日本語リソース、`hooks` 実装を照合しています。表示項目はAPK・プラン・LINEバージョンで変わる場合があります。

## 現在の設定項目（7件）

| 表示名 | 説明 | 初期 | 利用範囲 | 設定キー |
|---|---|:---:|---|---|
| VOOM アイコンを削除 | ボトムバーからVOOMタブを取り除きます。 | ON | Trial / Basic / Unlock | `remove_voom` |
| ミニアプリアイコンを削除 | ボトムバーからミニアプリタブを取り除きます。 | ON | Trial / Basic / Unlock | `remove_wallet` |
| ミニアプリ画面の項目を非表示 | ミニアプリ画面を中心としたレイアウト要素を検出し非表示にできます。先にミニアプリを開いて項目を読み込んでから設定してください。 | OFF | Trial / Basic / Unlock | `WalleteRemoveLayout` |
| ニュース・通話履歴アイコンを非表示 | ボトムバーからニュース・通話履歴タブを取り除きます。 | ON | Trial / Basic / Unlock | `remove_news_or_call` |
| ボトムバーのアイコンを均等配置 | タブを削除した後、残ったアイコンを均等に並べ直します。削除系の設定と組み合わせて使うと効果的です。 | ON | Trial / Basic / Unlock | `distribute_evenly` |
| ボトムバーのアイコンのタップ範囲を拡大 | ボトムバーのアイコンを押しやすくするため、タップできる範囲を広げます。 | ON | Trial / Basic / Unlock | `extend_clickable_area` |
| ボトムバーのラベルを非表示 | ボトムバーのアイコン下に表示される「トーク」「ニュース」などのテキストを非表示にします。すっきりした見た目になります。 | ON | Trial / Basic / Unlock | `remove_icon_labels` |

## LINE内の詳細設定

スイッチとは別に、機能が有効なときだけ表示される設定ボタンがあります。

- ミニアプリの項目非表示設定

## 変更後の確認

1. 設定を変更します。
2. 再起動確認が表示された場合はLINEを再起動します。
3. 対象画面で動作を確認します。問題があれば直前の設定をOFFに戻します。
