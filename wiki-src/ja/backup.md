# バックアップ・復元

[ホームへ戻る](../index.html) / [設定画面の使い方](settings-ui.md)

トーク履歴と関連データのバックアップ・復元を扱います。実行前に保存先を確認してください。

> 2026年6月21日時点の `LEINsOptions.java`、日本語リソース、`hooks` 実装を照合しています。表示項目はAPK・プラン・LINEバージョンで変わる場合があります。

## 現在の設定項目（3件）

| 表示名 | 説明 | 初期 | 利用範囲 | 設定キー |
|---|---|:---:|---|---|
| トーク履歴のバックアップ・復元 | rootなしでLINEのトーク履歴をバックアップ・復元できます。初回は LINE 内で初期設定が必要です。 | OFF | Trial / Basic / Unlock | `TalkBackUpRestore` |
| LEINsバックアップを上書きから保護 | LINEの公式クラウドバックアップがLEINsのバックアップを上書きしないよう保護します。 | OFF | Trial / Basic / Unlock | `TalkBackUpCloud_Disableoverride` |
| GMS Coreを利用してトークのバックアップ、復元を行います | この機能は試験的なため15.12.2のみを対象にしています。  LINEにGMS coreのパッチを当てる必要があります。  Micro G REの「アプリによるアカウントの検出を許可」を有効にしてください。  この機能でのトークのバックアップ/復元はLINE公式のデータに対して復元することを目的にしています | OFF | Basic / Unlock | `Revanced_LINE_Talk_BackUP_Restore` |

## LINE内の詳細設定

スイッチとは別に、機能が有効なときだけ表示される設定ボタンがあります。

- トーク履歴・チャットリスト・messageテーブル・画像フォルダのバックアップと復元
- MIDの確認

## 変更後の確認

1. 設定を変更します。
2. 再起動確認が表示された場合はLINEを再起動します。
3. 対象画面で動作を確認します。問題があれば直前の設定をOFFに戻します。
