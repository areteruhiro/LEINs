# LEINs の使い方・導入方法

[ホームへ戻る](../../docs/index.html)

LEINsでは、こちらのローカルパッチの利用をおすすめします。

## 1. パッチアプリをインストール

最初に、次のどちらかをインストールしてください。

- [NPatch（Google Drive）](https://drive.google.com/drive/folders/1tyi-4HomkELjDsHFHjKuXF5QOsSQuNhn?usp=drive_link)
- [LSPatch（GitHub Releases）](https://github.com/JingMatrix/LSPatch/releases/)

## 2. システムABIに合うLINEを入手

パッチアプリを開いて「HOME」を押し、表示された「システムABI」と同じABIのアプリをダウンロードしてください。

- [Google Drive](https://drive.google.com/drive/folders/1kHUdu7tEM-6gsowKhxo9b8e4ULrwQ7yF?usp=sharing)
- [MEGA](https://mega.nz/folder/TIkWTb5T#TPSj7x00vy4L-hqayxc7AQ)

## 3. 既存のLINEをバックアップ

パッケージ名を変更していないLINEを使う場合、既存のLINEとは署名が異なるため、そのまま上書きインストールできません。先にLINEをバックアップしてから既存のLINEをアンインストールし、ダウンロードしたアプリをインストールしてください。

## 4. 最初はスコープを選択しない

新しく導入する方は、最初はスコープを選択しないでください。

1. スコープ未選択のままLINEを起動する
2. LINEへログインし、利用できることを確認する
3. ログイン確認後に「LEINs」をスコープ選択する

## ローカルパッチを更新するとき

ローカルパッチの場合、更新はLEINsのみで大丈夫です。ただし、まれにスコープが外れる場合があります。更新後にLEINsが動作しないときは、スコープを確認してください。
