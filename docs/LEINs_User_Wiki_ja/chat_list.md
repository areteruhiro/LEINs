## 機能詳細

### チャットリスト内でのトークルームを作成ボタンの非表示 (`removeCreateChatButton`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: OFF
- **設定場所**: LEINs設定 > チャットリスト > チャットリストヘッダー
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: チャットリストにある新規トーク作成ボタンを非表示にします。
