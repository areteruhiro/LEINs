| 設定キー | 表示名 | 既定値 | プラン | 詳細設定 | 要確認 |
|---|---|---:|---|---|---|
| `remove_ads` | 広告を非表示 | ON | Trial / Basic / Unlock | なし | なし |
| `remove_premium_recommendation` | LYPプレミアムの広告を非表示 | ON | Trial / Basic / Unlock | なし | なし |

## 機能詳細

### 広告を非表示 (`remove_ads`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: ON
- **設定場所**: LEINs設定 > 広告
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: LINEの広告バナーを非表示にします。トークリストやホーム画面の広告に効果があります。

### LYPプレミアムの広告を非表示 (`remove_premium_recommendation`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: ON
- **設定場所**: LEINs設定 > 広告
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: LYPプレミアムへの加入を促すバナーやポップアップを非表示にします。

### おすすめを非表示 (`remove_recommendation`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: ON
- **設定場所**: LEINs設定 > 広告
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: LINEのホーム画面に表示されるサービスのおすすめカードを削除します。
