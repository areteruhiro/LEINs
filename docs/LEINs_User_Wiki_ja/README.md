# LEINs ユーザー向けWiki 日本語版

LEINsは、LINEの画面表示や通知、既読、バックアップなどを細かく調整するためのAndroid向け拡張モジュールです。このWikiでは、LEINsを初めて使う人でも「どこを開けばよいか」「どの設定をONにすればよいか」が分かるように、設定画面で表示される項目を中心に説明します。

専門用語や内部向けの名前はできるだけ言い換えています。まだ動作や対象プランを確認できていない機能は断定せず、[確認が必要な機能](unknowns.md) に残しています。

## 初めての人はここから

1. [はじめてのLEINs](getting-started.md) で、導入前の注意点と最初に見る場所を確認します。
2. [LEINs の版とプラン比較](editions.md) で、自分のAPKやプランで使える範囲を確認します。
3. [設定画面の使い方](settings-ui.md) で、LINE内のLEINs設定を開く流れを確認します。
4. 下のカテゴリ一覧から、使いたい目的に近いページを開きます。

## このWikiの読み方

- **設定キー** は内部名です。普通は覚えなくて大丈夫です。問い合わせやバグ報告で項目を特定するときに使います。
- **表示名** は設定画面に出る名前です。基本的にはこの名前で探してください。
- **既定値** は初期状態です。ONなら最初から有効、OFFなら自分で有効化します。
- **プラン** はその機能を使える版です。自分の画面に出ない場合は、プラン違いか、使っているAPKの種類が違う可能性があります。
- **詳細設定** が「あり」の場合、スイッチ以外に追加の設定画面やボタンがあります。
- **要確認** が「あり」の場合、説明がまだ完全ではありません。動作確認後に更新します。
