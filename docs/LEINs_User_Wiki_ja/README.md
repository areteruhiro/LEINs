# LEINs ユーザー向けWiki 日本語版

LEINsの設定画面で触れる機能を、ユーザー向けに説明するWikiです。設定画面で表示される項目を中心に、使い方と対象プランを整理しています。

不明な機能は断定しません。末尾の「確認が必要な機能」に質問として残し、回答をもらったものだけ各カテゴリページへ反映します。
