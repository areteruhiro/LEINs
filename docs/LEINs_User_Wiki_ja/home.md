# ホーム画面

[入口へ戻る](README.md) / [設定画面の使い方](settings-ui.md) / [確認が必要な機能](unknowns.md)

このページでは、LINEのホーム画面に出るサービス、ショッピング、ミニアプリなどの表示を整理する設定をまとめています。ホーム画面をすっきりさせたい場合に使います。

## 一覧

| 設定キー | 表示名 | 既定値 | プラン | 詳細設定 | 要確認 |
|---|---|---:|---|---|---|
| `remove_Services` | サービス項目を非表示 | OFF | Trial / Basic / Unlock | なし | なし |
| `RemoveCommerce` | ショッピング・コマース項目を非表示 | ON | Trial / Basic / Unlock | なし | なし |
| `RemoveMini` | ミニアプリ項目を削除 | OFF | Trial / Basic / Unlock | なし | なし |
| `RemoveProfileNotification` | 更新されたプロフィールを削除 | OFF | Trial / Basic / Unlock | なし | なし |
| `remove_service_labels` | サービスアイコンのラベルを非表示 | OFF | Trial / Basic / Unlock | なし | なし |

## 機能詳細

### サービス項目を非表示 (`remove_Services`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: OFF
- **設定場所**: LEINs設定 > ホーム画面 > ホーム画面
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: ホーム画面のサービス一覧にあるショートカットをすべて非表示にします。

### ショッピング・コマース項目を非表示 (`RemoveCommerce`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: ON
- **設定場所**: LEINs設定 > ホーム画面 > ホーム画面
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: LINE内のショッピング・コマース関連のボタンやバナーを非表示にします。

### ミニアプリ項目を削除 (`RemoveMini`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: OFF
- **設定場所**: LEINs設定 > ホーム画面 > ホーム画面
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: LINEの下部/ホームナビゲーションからミニアプリ項目を削除します。

### 更新されたプロフィールを削除 (`RemoveProfileNotification`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: OFF
- **設定場所**: LEINs設定 > ホーム画面 > ホーム画面
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: ホーム画面上部に表示される「プロフィールを更新しました」のバナーを非表示にします。

### サービスアイコンのラベルを非表示 (`remove_service_labels`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: OFF
- **設定場所**: LEINs設定 > ホーム画面 > ホーム画面
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: ホーム画面のサービスアイコン下にあるラベルテキストを非表示にします。
