## 機能詳細

### ブラウザアプリを使用 (`open_in_browser`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: OFF
- **設定場所**: LEINs設定 > ブラウザ・リンク > ブラウザ・その他
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: 上の「外部ブラウザで開く」が有効なとき、ブラウザタブではなく独立したブラウザアプリとして開きます。

### リンクを外部ブラウザで開く (`redirect_webview`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: ON
- **設定場所**: LEINs設定 > ブラウザ・リンク > ブラウザ・その他
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: LINE内のリンクをタップすると、LINE内蔵ブラウザではなくスマホの標準ブラウザで開きます。

### 設定を LINE に埋め込まない (`removeOption`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: OFF
- **設定場所**: LEINs設定 > ブラウザ・リンク > ブラウザ・その他
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: LINE内に表示されるLEINsの歯車ボタンを非表示にします。LEINsアプリからは引き続き設定を変更できます。
