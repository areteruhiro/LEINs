## 機能詳細

### 通知にコピーアクションを追加 (`AddCopyAction`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: OFF
- **設定場所**: LEINs設定 > 通知 > 通知拡張
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: メッセージ通知に「コピー」ボタンを追加します。LINEを開かずに通知からテキストをコピーできます。

### 特定のユーザー名、グループ名のみ以外の通知をオフ (`CansellNotification`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: OFF
- **設定場所**: LEINs設定 > 通知 > 通知フィルター
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **詳細設定**: `特定のユーザー名、グループ名のみ以外の通知をオフ` がONのとき、同名のボタンから通知を許可するユーザーやグループを設定できます。
- **機能説明**: 指定した連絡先・グループからの通知だけを受け取り、それ以外は全て通知をキャンセルします。LINE内のLEINsボタンからリストを管理できます。

### 着信通知に返信ボタンの作成 (`CreateCallReply`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: OFF

- **機能説明**: 着信通知に返信アクションを追加します。

- **初期状態**: ON

### 特定グループの通知をミュート (`Disabled_Group_notification`)

- **機能説明**: 指定したグループからの通知をオフにします。LINE内のLEINsボタンからミュートリストを管理できます。
