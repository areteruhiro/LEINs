# 通話・着信音

[ホームへ戻る](../index.html) / [設定画面の使い方](settings-ui.md)

このページでは、LINE通話のボタン、着信音、発信音に関する設定をまとめています。音や通話画面の動作は端末やLINEバージョンの影響を受けやすいため、変更後は実際に発着信して確認してください。

## 一覧

| 設定キー | 表示名 | 既定値 | プラン | 詳細設定 | 要確認 |
|---|---|---:|---|---|---|
| `CallComing` | 通話中は着信設定を自動で切り替え | OFF | Trial / Basic / Unlock | なし | なし |
| `CallDisconnectTone` | 通話終了時に効果音を再生(非root要) | OFF | Trial / Basic / Unlock | なし | なし |
| `CallOpenApplication` | 通話画面にLINEを開くボタンを追加 | ON | Trial / Basic / Unlock | なし | なし |
| `callTone` | 本体の発着信音を鳴らす（非root） | OFF | Trial / Basic / Unlock | なし | なし |
| `DialTone` | 発信音を無効化 | OFF | Trial / Basic / Unlock | なし | なし |
| `MuteTone` | LINE着信音をミュート (root) | OFF | Trial / Basic / Unlock | なし | なし |
| `Original_Tone` | オリジナルの着信音に変更 | OFF | Trial / Basic / Unlock | なし | なし |
| `ringtonevolume` | (発信音を鳴らす)着信音を利用する | OFF | Trial / Basic / Unlock | なし | なし |
| `SilentCheck` | SilentCheck | OFF | Trial / Basic / Unlock | なし | なし |
| `StopCallTone` | 着信音 停止ボタンを追加 | OFF | Trial / Basic / Unlock | なし | なし |

## 機能詳細

### 通話中は着信設定を自動で切り替え (`CallComing`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: OFF
- **設定場所**: LEINs設定 > 通話・着信音 > 通話UI
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: 設定→通話→通話の着信許可の有無を通話開始/切断で変更します

### 通話終了時に効果音を再生(非root要) (`CallDisconnectTone`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: OFF
- **設定場所**: LEINs設定 > 通話・着信音 > 着信音
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: 通話終了時に効果音（lineapp_endthis_16k.wav）を再生します。LEINsの設定フォルダにファイルを置いてください。

### 通話画面にLINEを開くボタンを追加 (`CallOpenApplication`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: ON
- **設定場所**: LEINs設定 > 通話・着信音 > 通話UI
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: 通話画面にLINEアプリを開くボタンを追加します。通話中にチャットを確認したいときに便利です。

### 本体の発着信音を鳴らす（非root） (`callTone`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: OFF
- **設定場所**: LEINs設定 > 通話・着信音 > 着信音
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: LINE通話の着信音を、LINEの内蔵音ではなくスマホのデフォルト着信音に変更します。非rootユーザー向けの機能です。

### 発信音を無効化 (`DialTone`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: OFF
- **設定場所**: LEINs設定 > 通話・着信音 > 着信音
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: 電話をかけているときに鳴るダイアルトーンを無効にします。

### LINE着信音をミュート (root) (`MuteTone`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: OFF
- **設定場所**: LEINs設定 > 通話・着信音 > 着信音
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: LINEの着信音を完全に無音にします。rootユーザー向けの機能です。

### オリジナルの着信音に変更 (`Original_Tone`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: OFF
- **設定場所**: LEINs設定 > 通話・着信音 > 着信音
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: LINEの着信音をシステムの着信音ピッカーで選択した音に変更します。

### (発信音を鳴らす)着信音を利用する (`ringtonevolume`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: OFF
- **設定場所**: LEINs設定 > 通話・着信音 > 着信音
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: LINE通話の着信・発信音量として端末の着信音量を使用します。

### SilentCheck (`SilentCheck`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: OFF
- **設定場所**: LEINs設定 > 通話・着信音 > 通話UI
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: 着信音を鳴らす前にスマホがサイレントモードかどうかを確認し、サイレント中は着信音をスキップします。

### 着信音 停止ボタンを追加 (`StopCallTone`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: OFF
- **設定場所**: LEINs設定 > 通話・着信音 > 着信音
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: 非rootユーザー向けの着信音再生機能に「停止ボタン」を追加します。
