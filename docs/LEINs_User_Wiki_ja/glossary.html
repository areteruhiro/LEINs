LEINsの説明でよく出る言葉を、初めての人向けにまとめます。

用語 意味
LEINs LINEに機能追加や表示調整を行うAndroid向け拡張モジュールです。
APK Androidアプリのインストールファイルです。配布名によってTrial版や通常版を見分けます。
ROOT Android端末の管理者権限を使う導入方法です。分からない場合は導入案内をよく確認してください。
LsPatch アプリをパッチして使う導入方法です。ROOTなしで使う場合の選択肢になります。
プラン Trial / Basic / Unlock の利用範囲です。プランによって表示される設定項目が変わります。
設定キー 開発・問い合わせ用の内部名です。ユーザーは表示名を見れば大丈夫です。
既定値 初期状態のON/OFFです。
詳細設定 スイッチだけではなく、別画面で細かく調整する設定です。
親子関係 ある機能をONにしたときだけ、関連する子機能が使える関係です。
再起動 LINEアプリを閉じて開き直すことです。設定反映に必要な場合があります。
要確認 動作説明や対象プランがまだ確定していない項目です。
上級者向け 通信、トークン、内部ファイルなど、仕組みを理解している人向けの項目です。
