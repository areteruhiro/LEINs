# ボタン設定

[ホームへ戻る](../index.html) / [設定画面の使い方](settings-ui.md)

このページでは、LEINsが追加するボタンや補助ボタンの位置・表示方法をまとめています。ボタンが押しにくい、画面に重なる、場所を変えたいときに使います。

## 一覧

| 設定キー | 表示名 | 既定値 | プラン | 詳細設定 | 要確認 |
|---|---|---:|---|---|---|
| `FontChange` | アプリ全体を任意のフォントに変更します | ON | Unlock版のみ | なし | なし |
| `FontChange_Liff_Web` | LiffアプリとLINE内Webのフォントを変更します | ON | Unlock版のみ | なし | なし |
| `photoboothButtonOption` | LINE通話ボタンを非表示 | ON | Trial / Basic / Unlock | なし | なし |
| `videoButtonOption` | ビデオ通話ボタンを非表示（グループ） | ON | Trial / Basic / Unlock | なし | なし |
| `videoSingleButtonOption` | ビデオ通話ボタンを非表示（個人） | ON | Trial / Basic / Unlock | なし | なし |
| `voiceButtonOption` | 音声通話ボタンを非表示 | OFF | Trial / Basic / Unlock | なし | なし |

## カテゴリ共通の詳細操作
- **ボタンの設定**: LEINsボタンの位置や余白を調整できます。

## 機能詳細

### アプリ全体を任意のフォントに変更します (`FontChange`)

- **利用できるプラン**: Unlock版のみ
- **初期状態**: ON
- **設定場所**: LEINs設定 > ボタン設定 > ボタン設定
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: 設定したディレクトリに\"lein.ttf\"ファイルを配置してください。\nフォントのメトリクス（行の高さ）を手動調整する場合は\"font.txt\"を編集し ascent=- / descent= の形式で編集して下さい

### LiffアプリとLINE内Webのフォントを変更します (`FontChange_Liff_Web`)

- **利用できるプラン**: Unlock版のみ

- **機能説明**: 設定したディレクトリに\"lein.ttf\"ファイルを配置してください。\nフォントのメトリクス（行の高さ）を手動調整する場合は\"font.txt\"を編集し ascent=- / descent= の形式で編集して下さい

- **機能説明**: トーク内の通話メニューからLINE通話ボタンを非表示にします。
