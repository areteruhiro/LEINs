LEINs の版とプラン比較

ホームへ戻る / はじめてのLEINs / 設定画面の使い方

このページでは、最低限の機能を使える Trial版 と、有料プランとして扱う Basic版 / Unlock版 の違いを説明します。

先に確認すること

LEINsには、APKの種類と利用プランの両方があります。初めて使う場合は、まず自分が入れたAPK名と、設定画面に表示されるプランを確認してください。

確認するもの 見る理由
APK名 Trial版か通常版かを見分けるため
設定画面に出る項目 自分の環境で使える機能を確認するため
プラン表記 Basic / Unlock の対象機能かを判断するため

Wikiに載っているのに設定画面に表示されない機能は、プラン違い、APK違い、または現在のLINEバージョンでは無効化されている可能性があります。

位置づけ

種別 位置づけ 想定ユーザー 提供内容
Trial版 最低限の機能だけを使える版 基本機能だけを使いたいユーザー 日常利用向けの基本機能に限定
Basic版 有料版 LEINsの通常機能を広く使いたいユーザー Basic向け機能を利用可能
Unlock版 サブスク版 すべてのLEINs機能を使いたいユーザー Unlock専用機能を含む全機能を利用可能

Trial版は別アプリとして共存させる版ではありません。通常版とは別のAPKとして配布されますが、同じアプリとして扱われます。

Trial版で使える機能

Trial版は、日常利用に必要な最低限の機能と、その動作に必要な基盤機能に限定します。

分類 Trial版で使える機能
広告・おすすめ非表示 広告削除、おすすめ削除、GCS/LYPおすすめ削除、ホーム/サービス/ミニアプリ周辺の整理
タブ・ホーム・ヘッダー整理 VOOM/ニュース/ミニアプリ系タブ非表示、タブ文字非表示、タップ範囲拡張、Agent i/OpenChat/AI系ボタン非表示、ヘッダー整理
既読制御・既読履歴 既読防止、既読履歴、既読確認、既読関連の補助ボタン、クイック切替
送信取消保護 送信取消防止、送信取消メニュー、送信取消可能時間の延長
ブラウザ 外部ブラウザ起動、WebViewリダイレクト
フォント カスタムフォント
画像・動画品質 高画質画像、動画送信制限回避に相当する機能
検索補助 1文字検索、検索フィルター
通知 リアクション通知、通知ミュート削除、サイレント送信関連
FCM/常駐 foreground service化に相当する通知不達対策
リアクション一覧 リアクション一覧表示、リアクションDB連携
表示名 ローカル名
基盤 設定画面埋め込み、モジュール導入確認、リソースブリッジ

Trial版では使えない機能

Trial版では、最低限の利用範囲を超えるLEINs独自機能や、Unlock/開発者向けの機能は含めません。

分類 Trial版では使えない例
Unlock/解除系 LYP Premium解除、スタンプTrial解除、テーマ無料化、サブ端末ゲート解除
LEINs独自バックアップ LEINs独自バックアップ、GMSCoreバックアップ復元、バックアップ上書き保護
開発者・解析向け 通信ログ出力、HEX出力、Token取得、設定項目フィルター、年齢確認スキップ、生体情報偽装
通話・着信音拡張 通話ボタン追加、着信音変更、発信音制御、通知返信などの通話系拡張
テーマ改変 ピュアダーク化、テーマストア関連、テーマ検証回避
自動化・改変系 自動返信、予約送信、メッセージ追加、アルバム自動ダウンロードなど

機能の範囲 基本機能のみ Basic向け機能まで Unlock専用機能を含む

配布時の見分け方

版 APK名の目安
Trial版 Trial版として区別できる名前
Basic/Unlockを含む通常版 LEINsの通常版として区別できる名前

配布ページでは、Trial版と通常版のAPK名が分かれていることを確認してください。Basic版とUnlock版は同じ通常版APK内で、利用できる機能がプランに応じて変わります。
