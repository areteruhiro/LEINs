## 機能詳細

### アルバム自動ダウンロード (`AlbumAutoDownload`)

- **利用できるプラン**: Basic / Unlock (Trialでは非表示)
- **初期状態**: OFF
- **設定場所**: LEINs設定 > チャット・メッセージ > その他
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: アルバムの画像や動画を自動でダウンロードします

### 非表示チャットを再表示されないようにする (`Archived_message`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: OFF
- **設定場所**: LEINs設定 > チャット・メッセージ > チャットリスト
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: 手動で非表示にしたチャットが、自動的にメインのチャットリストに戻ってこないようにします。

### アーカイブ設定 (`Archived_Stting`)

- **利用できるプラン**: Trial / Basic / Unlock

- **詳細設定**: `アーカイブ設定` がONのとき、`アーカイブ設定` ボタンから非表示チャットの再表示防止期間を設定できます。
- **機能説明**: 非表示チャットの再表示防止期間を設定します
