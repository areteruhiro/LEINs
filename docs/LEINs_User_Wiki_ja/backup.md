## カテゴリ共通の詳細操作
- **バックアップ**: トーク履歴のバックアップを開始します。
- **復元**: バックアップファイルを選んでトーク履歴を復元します。
- **PC版_messageテーブルを復元**: PC版のメッセージデータを選んで復元します。
- **チャットリストを復元**: チャットリストのバックアップを選んで復元します。
- **トーク画像フォルダのバックアップを開始**: トーク画像フォルダをバックアップします。
- **トーク画像のリストア**: バックアップしたトーク画像を復元します。
- **Mid IDを取得**: 自分のMIDを確認します。

## 機能詳細

### GMS Coreを利用してトークのバックアップ、復元を行います (`Revanced_LINE_Talk_BackUP_Restore`)

- **利用できるプラン**: Basic / Unlock (Trialでは非表示)
- **初期状態**: OFF
- **設定場所**: LEINs設定 > バックアップ・復元 > LEINsバックアップ
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: この機能は試験的なため15.12.2のみを対象にしています。  LINEにGMS coreのパッチを当てる必要があります。  この機能でのトークのバックアップ/復元はLINE公式のデータに対して復元することを目的にしています

### LEINs独自バックアップ（実験的機能） (`LEINsTalkBackUpRestore`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: OFF
- **設定場所**: LEINs設定 > バックアップ・復元 > LEINsバックアップ
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: LEINs独自の実験的なバックアップ機能です。使用前に「LEINsバックアップを上書きから保護」を必ず有効にしてください。

### LEINsバックアップを上書きから保護 (`TalkBackUpCloud_Disableoverride`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: OFF
- **設定場所**: LEINs設定 > バックアップ・復元 > LEINsバックアップ
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: LINEの公式クラウドバックアップがLEINsのバックアップを上書きしないよう保護します。

### トーク履歴のバックアップ・復元 (`TalkBackUpRestore`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: OFF
- **設定場所**: LEINs設定 > バックアップ・復元 > LEINsバックアップ
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: rootなしでLINEのトーク履歴をバックアップ・復元できます。初回は LINE 内で初期設定が必要です。
