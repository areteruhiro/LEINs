[入口へ戻る](README.md) / [設定画面の使い方](settings-ui.md) / [確認が必要な機能](unknowns.md)

## カテゴリ共通の詳細操作
- **リクエストを改変**: 通信リクエストの改変設定を開きます。
- **レスポンスを改変**: 通信レスポンスの改変設定を開きます。
- **動的検出クラスをリセット**: 自動検出された内部情報を初期化します。
- **ライセンスキーをリセット**: 保存済みのライセンスキーを初期化します。
- **アクセストークン設定**: アクセストークンを設定します。
- **設定URIをリセット**: 設定用URIを初期化します。
- **LEINs内部のファイルをコピー**: LEINsが使用するファイルを別の場所へコピーします。
- **寄付者一覧**: 寄付者一覧を表示します。
- **更新内容一覧**: 更新内容の一覧を表示します。

## 機能詳細

### 年齢確認をスキップ (`AgeCheckSkip`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: OFF
- **設定場所**: LEINs設定 > 上級者向け > 高度な設定
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: 現在この機能は機能していません

### LEINsの更新を自動確認 (`AutoUpDateCheck`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: OFF
- **設定場所**: LEINs設定 > 上級者向け > アプリの動作
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: LEINsのGitHubページを自動確認し、新しいバージョンがあれば通知します。

### 生体情報連携を許可をLSpach/NPachでも可能に (`Biometric_information`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: OFF
- **設定場所**: LEINs設定 > 上級者向け > LSPatch / 非rootユーザー向け
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: Enables fingerprint and face unlock for LINE when running on a non-root setup.

### ⚠ E2EE暗号化を無効化 (`E2EE_Disable`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: OFF
- **設定場所**: LEINs設定 > 上級者向け > セキュリティ
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: ⚠ 危険: エンドツーエンド暗号化を無効化します。自分だけでなく、相手側の暗号化も解除される可能性があります。リスクを完全に理解した上でのみ使用してください。

### サブ端末をメイン端末に偽装 (`FiASubDeviceGateFalse`)

- **利用できるプラン**: Unlock版のみ
- **初期状態**: OFF
- **設定場所**: LEINs設定 > 上級者向け > セキュリティ
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: LINEのショップ用サブ端末判定をfalseに固定し、サブ端末をメイン端末に偽装します。

### SHA-1 HEXログ出力 (`HexOutput`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: OFF
- **設定場所**: LEINs設定 > 上級者向け > LSPatch / 非rootユーザー向け

- **機能説明**: FCMトークンを取得が上手に出来ない場合に有効にします。

### 通信内容をログに出力 (開発者用) (`output_communication`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: OFF
- **設定場所**: LEINs設定 > 上級者向け > ログ・通信
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: LINEが送受信するすべての通信内容をログに記録します。デバッグ・開発用途向けです。

### LINE設定項目フィルター (`Setting_Filter`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: OFF
- **設定場所**: LEINs設定 > 上級者向け > LSPatch / 非rootユーザー向け
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **詳細設定**: `LINE設定項目フィルター` がONのとき、`LINE設定項目フィルター` ボタンから表示する設定項目を編集できます。
- **機能説明**: LINEのメイン設定に表示する項目を選択できます。新しく検出された項目は自動で追加されます。

### フォアグラウンド通知の発行 (`SystemForegroundService`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: OFF
- **設定場所**: LEINs設定 > 上級者向け > アプリの動作
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: LINEをフォアグラウンドサービスとして常時起動し、LINEを開いていなくても通知が届くようにします。バッテリー消費が増える場合があります。

### FCMトークンを取得 (`TokenGet`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: OFF
- **設定場所**: LEINs設定 > 上級者向け > LSPatch / 非rootユーザー向け

- **詳細設定**: `FCMトークンを取得` がONのとき、`FCMトークンを登録` ボタンから通知用トークンを登録できます。
- **機能説明**: LINEのログイントークンをログに記録します。外部ツールと連携する上級者向けの機能です。

### LYP Premium機能の解除 (`LypPremiumUnlock`)

- **利用できるプラン**: Unlock版のみ
- **初期状態**: OFF
- **設定場所**: LEINs設定 > 上級者向け > 解除
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: LYPのアプリアイコン機能のみを解除します。 ただし、下記の理由があるため最終説明は要確認です。
- **要確認**: 説明と対象プランを確認してください
