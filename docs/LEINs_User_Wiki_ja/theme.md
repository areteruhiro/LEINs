## 機能詳細

### 「ブラック」テーマをピュアダークに変換 (`Dark_Theme_Change`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: OFF
- **設定場所**: LEINs設定 > テーマ・外観 > テーマ・外観
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: LINEの「ブラック」テーマをピュアブラックに変換します。使用前にLINEの設定で「ダークモードでブラックを適用」をオフにしてください。

### ダークテーマをピュアダークにする (`DarkColor`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: OFF
- **設定場所**: LEINs設定 > テーマ・外観 > テーマ・外観
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: ナビゲーションボタンを純黒にし、ダークモードの背景を真っ黒（#000000）にします。

### ダークモードをシステムと同期 (`DarkModSync`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: ON
- **設定場所**: LEINs設定 > テーマ・外観 > テーマ・外観
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: スマホのシステムのダーク/ライトモード設定に合わせて、LINEのダークモードを自動で切り替えます。

### 着せ替えの検証を無効 (`Enable_Theme_Validation`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: OFF
- **設定場所**: LEINs設定 > テーマ・外観 > テーマ・外観
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: LINEの内部テーマ検証チェックを再有効化します。通常はカスタムテーマを使うために無効化されています。

### ダークテーマに着せ替え (`WhiteToDark`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: OFF
- **設定場所**: LEINs設定 > テーマ・外観 > テーマ・外観
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: LINEのダークテーマを強制的に適用します。
