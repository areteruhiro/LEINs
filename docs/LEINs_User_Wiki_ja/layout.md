## 機能詳細

### ボトムバーのアイコンを均等配置 (`distribute_evenly`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: ON
- **設定場所**: LEINs設定 > 画面レイアウト・UI > ボトムバー
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: タブを削除した後、残ったアイコンを均等に並べ直します。削除系の設定と組み合わせて使うと効果的です。

### ボトムバーのアイコンのタップ範囲を拡大 (`extend_clickable_area`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: ON
- **設定場所**: LEINs設定 > 画面レイアウト・UI > ボトムバー

- **機能説明**: ボトムバーのアイコンを押しやすくするため、タップできる範囲を広げます。

- **機能説明**: ボトムバーのアイコン下に表示される「トーク」「ニュース」などのテキストを非表示にします。すっきりした見た目になります。

### ミニアプリ画面の項目を非表示 (`WalleteRemoveLayout`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: OFF
- **設定場所**: LEINs設定 > 画面レイアウト・UI > ボトムバー
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **詳細設定**: `ミニアプリ画面の項目を非表示` がONのとき、`ミニアプリの項目非表示設定` ボタンから隠す項目を選べます。
- **機能説明**: ミニアプリ画面を中心としたレイアウト要素を検出し非表示にできます。先にミニアプリを開いて項目を読み込んでから設定してください。
