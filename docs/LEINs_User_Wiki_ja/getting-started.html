はじめてのLEINs

初めて開いたときに、まず覚えておきたい流れです。

使い始めるまで

LINEを開く

いつもどおりLINEを開きます。

LEINsを開く

LINE内に追加されたLEINsのボタンを押します。

目的を選ぶ

一覧から「広告」「通知」「既読」など、変えたい内容に近い項目を選びます。

スイッチを切り替える

使いたいものをオンにします。変化が見えないときはLINEを開き直してください。

最初に試しやすいこと

画面をすっきりさせる

使わないタブやおすすめ表示を減らせます。

通知を見やすくする

不要な通知を減らしたり、通知から操作しやすくしたりできます。

既読の付き方を選ぶ

既読を付けたくない場面や、あとで読み返したい場面に合わせられます。

大切なトークや写真がある場合は、設定を大きく変える前にバックアップしてください。
