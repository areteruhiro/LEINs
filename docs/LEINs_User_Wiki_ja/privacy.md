## 機能詳細

### >ブロック監視[誤検知可能性有] (`BlockCheck`)

- **利用できるプラン**: Basic / Unlock (Trialでは非表示)
- **初期状態**: ON
- **設定場所**: LEINs設定 > プライバシー・既読 > ブロック確認
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **詳細設定**: `>ブロック監視[誤検知可能性有]` がONのとき、同名のボタンから監視対象を設定できます。
- **機能説明**: ブロックされた可能性のあるユーザーの名前を変更します

### チャットリストに「送信したら既読」切り替えボタンを追加 (`Keep_unread_PopupListView`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: OFF
- **設定場所**: LEINs設定 > プライバシー・既読 > 既読管理
- **使い方**: この項目のスイッチをON/OFFします。変更後に反映されない場合は、設定画面下部の再起動ボタン、または表示される再起動確認からLINEを再起動します。
- **機能説明**: チャットリストのヘッダーに送信したら既読ボタンを追加します。長押しメニューを開いた後も未読のまま維持できます。

### 「送信したら既読」スイッチを表示（非root） (`Keep_UnreadLSpatch`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: OFF

- **機能説明**: rootなしでLINEを使っているユーザー向けに「未読にする」スイッチを表示します。

### (常に既読を付けない)既読を付ける相手の編集 (`PreventMarkAsRead_Setting`)

- **利用できるプラン**: Trial / Basic / Unlock
- **初期状態**: OFF

- **機能説明**: 特定のチャットだけ既読送信を個別に制御します。「常に未読のまま」と併用できます。
