ダーク表示、着せ替え検証、テーマストアの表示を調整します。

2026年6月21日時点の LEINsOptions.java、日本語リソース、hooks 実装を照合しています。表示項目はAPK・プラン・LINEバージョンで変わる場合があります。

現在の設定項目（7件）

表示名 説明 初期 利用範囲 設定キー
ダークテーマをピュアダークにする ナビゲーションボタンを純黒にし、ダークモードの背景を真っ黒（#000000）にします。 OFF Trial / Basic / Unlock DarkColor
ダークモードをシステムと同期 スマホのシステムのダーク/ライトモード設定に合わせて、LINEのダークモードを自動で切り替えます。 ON Trial / Basic / Unlock DarkModSync
「ブラック」テーマをピュアダークに変換 LINEの「ブラック」テーマをピュアブラックに変換します。使用前にLINEの設定で「ダークモードでブラックを適用」をオフにしてください。 OFF Trial / Basic / Unlock Dark_Theme_Change
ダークテーマに着せ替え LINEのダークテーマを強制的に適用します。 OFF Trial / Basic / Unlock WhiteToDark
テーマ無料化 "着せ替えの検証を無効"を有効にして下さい。 OFF Basic / Unlock Theme_free
サブ端末でもテーマストアを表示 サブ端末ログイン時でも、LINEのテーマストア項目とボタンを表示します。 OFF Basic / Unlock SecondaryThemeStore
着せ替えの検証を無効 LINEの内部テーマ検証チェックを再有効化します。通常はカスタムテーマを使うために無効化されています。 OFF Trial / Basic / Unlock Enable_Theme_Validation

LINE内の詳細設定

スイッチとは別に、機能が有効なときだけ表示される設定ボタンがあります。

変更後の確認

  1. 設定を変更します。
  2. 再起動確認が表示された場合はLINEを再起動します。
  3. 対象画面で動作を確認します。問題があれば直前の設定をOFFに戻します。
