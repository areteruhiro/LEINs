# 確認が必要な機能

[入口へ戻る](README.md) / [はじめてのLEINs](getting-started.md)

ここにある項目は、説明文や対象プランをまだ確定していないものです。推測で本文へ断定記載せず、確認できたものだけ各カテゴリページへ反映します。

初めて使う人は、このページの機能を無理に有効化しなくても大丈夫です。説明が確定しているカテゴリページの機能から試してください。

確認できた場合は、次の情報があると更新しやすくなります。

- 設定画面に表示される名前
- 実際にON/OFFしたときの変化
- 使っているLEINsのAPK種類とプラン
- LINEのバージョン
- 再起動後も同じ動作かどうか

| 表示名 | 確認したい内容 |
|---|---|
| 送信取り消しを強制的に24時間に変更 | ユーザー向け説明、実際の使い方、対象プランを確認してください。 |
| Agent i を非表示 | ユーザー向け説明、実際の使い方、対象プランを確認してください。 |
| 既読ボタンを押した際に全ての通知を削除する | ユーザー向け説明、実際の使い方、対象プランを確認してください。 |
| LYP Premium機能の解除 | ユーザー向け説明、実際の使い方、対象プランを確認してください。 |
| 既読にする | ユーザー向け説明、実際の使い方、対象プランを確認してください。 |
