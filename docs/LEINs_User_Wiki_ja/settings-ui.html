設定画面の使い方

探す、選ぶ、切り替える。この3つだけ覚えれば使えます。

画面の見方

検索

画面上の検索欄に「通知」「広告」「既読」などを入力すると、目的の設定を早く見つけられます。

カテゴリ

似た設定がまとまっています。迷ったときは、いちばん近い名前のカテゴリから開いてください。

オン・オフ

スイッチが緑色ならオン、灰色ならオフです。

戻る

設定が終わったら「戻る」を押してLINEへ戻ります。

購入方法

購入するを押す

ライセンス認証の画面で「購入する」を押すと、購入プランの選択画面が開きます。

プランを選ぶ

7日お試し、月額、年額、買い切りから選べます。

支払いを完了する

開いた購入ページで案内に沿って支払いを進めます。

認証する

購入後に届いたライセンスキーを入力し、「認証」を押します。

7日お試し

0円

まず動作を確認したい方向けです。

月額プラン

250円 / 月

必要な期間だけ使いたい方向けです。

年額プラン

2,600円 / 年

継続して使う方向けです。

買い切り

2,500円

一度の購入で継続して使いたい方向けです。

表示金額は2026年6月21日に購入ページで確認した金額です。購入前に表示される最終金額も確認してください。

うまく変わらないとき

LINEを開き直す

一度LINEを閉じて、もう一度開くと反映されることがあります。

ひとつずつ試す

複数を一度に変えず、ひとつずつ確認すると元に戻しやすくなります。

元に戻す

表示がおかしくなった場合は、直前にオンにした設定をオフにしてください。
