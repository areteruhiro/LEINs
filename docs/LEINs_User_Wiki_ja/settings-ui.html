設定画面の使い方

探す、選ぶ、切り替える。この3つだけ覚えれば使えます。

画面の見方

検索

画面上の検索欄に「通知」「広告」「既読」などを入力すると、目的の設定を早く見つけられます。

カテゴリ

似た設定がまとまっています。迷ったときは、いちばん近い名前のカテゴリから開いてください。

オン・オフ

スイッチが緑色ならオン、灰色ならオフです。

戻る

設定が終わったら「戻る」を押してLINEへ戻ります。

うまく変わらないとき

LINEを開き直す

一度LINEを閉じて、もう一度開くと反映されることがあります。

ひとつずつ試す

複数を一度に変えず、ひとつずつ確認すると元に戻しやすくなります。

元に戻す

表示がおかしくなった場合は、直前にオンにした設定をオフにしてください。
