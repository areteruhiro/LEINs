プライバシー・既読15項目

既読管理

メッセージを常に未読のままにするメッセージを開いても既読を送信しません。

既読を付ける相手の編集相手ごとに既読送信を設定します。

チャットを開くと未読バッジを消す端末側の未読表示だけを消します。

クイック切替を追加トーク一覧のメニューから切り替えます。

既読チェッカー
