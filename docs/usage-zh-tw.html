安裝指南

使用方法

LEINs 建議使用此處提供的本機修補版本。依照以下順序操作即可開始使用。

步驟 1

安裝修補工具

請先安裝 NPatch 或 LSPatch 其中之一。

推薦 NPatch 從 Google Drive 開啟 另一個選擇 LSPatch 開啟 GitHub Releases

步驟 2

下載符合系統 ABI 的 LINE

開啟修補工具並點選「HOME」，然後下載與畫面上「系統 ABI」相同 ABI 的應用程式。

下載 Google Drive 開啟已修補的 LINE 下載 MEGA 開啟已修補的 LINE

步驟 3

備份現有的 LINE 資料

使用未變更套件名稱之 LINE 的使用者

下載的應用程式與現有 LINE 的簽章不同，因此無法直接覆蓋安裝。請先備份聊天記錄與其他重要資料，解除安裝現有的 LINE，然後再安裝下載的應用程式。

步驟 4

第一次請先不要選擇作用域

初次安裝時，請依照以下順序操作。

  1. 1
    不選擇作用域直接啟動

    先不要選擇作用域，直接啟動 LINE。

  2. 2
    登入 LINE

    正常登入，並確認 LINE 可以使用。

  3. 3
    將 LEINs 加入作用域

    確認登入成功後，請在作用域中選擇「LEINs」。

確認可以登入之前，請不要選擇作用域。
使用本機修補版本時，只需要更新 LEINs。不過，作用域偶爾可能會被取消；如果更新後 LEINs 無法運作，請檢查作用域設定。
