Setup guide

How to use

We recommend using one of the locally patched builds listed here with LEINs. Follow these steps in order to get started.

Step 1

Install a patching app

First, install either NPatch or LSPatch.

Recommended NPatch Open in Google Drive Another option LSPatch Open GitHub Releases

Step 2

Download LINE for your system ABI

Open the patching app and tap “HOME.” Download the app whose ABI matches the “System ABI” shown there.

Download Google Drive Open the patched LINE builds Download MEGA Open the patched LINE builds

Step 3

Back up your current LINE data

If you use LINE without changing its package name

The downloaded app has a different signature from the existing LINE app, so it cannot be installed as an update. Back up your chats and other important data first, uninstall the existing LINE app, and then install the downloaded app.

Step 4

Do not select a scope at first

For a new installation, follow these steps in order.

  1. 1
    Start without selecting a scope

    Launch LINE without selecting a scope.

  2. 2
    Sign in to LINE

    Sign in normally and confirm that LINE works.

  3. 3
    Select LEINs in the scope

    After confirming that you can sign in, select “LEINs” in the scope.

Do not select a scope before confirming that you can sign in.
When using a local patch, you only need to update LEINs. However, the scope may occasionally be cleared. If LEINs does not work after an update, check the scope setting.
