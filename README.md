# <img src="app/src/main/ic_launcher-playstore.png" width="60px"> LEINs

## 概要
**「LINEをあなたの使い方に合わせて」**

LEINs は、LINE アプリ体験を向上させるための  
ユーザー向けカスタマイズツールです。

表示の調整、通知の最適化、ローカルデータの管理など、  
**ユーザー自身の端末・データに対する操作性**を向上させます。

本ツールはクライアント側のUI・表示設定のみを対象としており、  
外部サーバーや他サービスの動作・認証に関与するものではありません。

> 本プロジェクトは独立した開発によるものであり、特定のサービスとは関係ありません。

## ✨ 主な機能
- **UI（表示）のカスタマイズ** — フォント・レイアウト・カラーテーマなど端末表示の調整  
- **通知・表示の最適化** — ユーザーの好みに合わせた通知スタイルの設定  
- **ローカルデータのバックアップ / リストア** — 端末内データの自己管理支援  
- **操作性の補助機能** — ユーザビリティ向上のためのショートカット・アクセシビリティ改善  

詳細はこちら：  
https://github.com/areteruhiro/LEINs/blob/master/FunctionLIST.md

## 💾 バックアップ / リストア
LEINs は、端末内のローカルデータを安全に管理するための機能を提供します。

手順：
1. バックアップフォルダを安全な場所へ移動  
2. アプリ内でバックアップを実行  
3. 必要に応じてリストア機能を使用  

※実行前にデータのバックアップを行ってください
