# <img src="app/src/main/ic_launcher-playstore.png" width="60px"> LEINs

## 概要
**「メッセージ環境をより快適に」**

LEINs は、Android上のメッセージ利用体験を向上させるための  
カスタマイズおよびユーティリティツールです。

ユーザーがアプリの操作性や表示、データ管理を柔軟に調整できるよう、  
各種補助機能を提供します。

> 本プロジェクトは独立した開発によるものであり、特定のサービスとは関係ありません。

## 💾 バックアップ / リストア
LEINs では、ローカルデータの管理機能を提供しています。

基本手順：
1. バックアップフォルダを安全な場所へ移動  
2. アプリ内でバックアップを実行  
3. 必要に応じてリストア機能を使用  

※実行前に必ずデータのバックアップを行ってください
